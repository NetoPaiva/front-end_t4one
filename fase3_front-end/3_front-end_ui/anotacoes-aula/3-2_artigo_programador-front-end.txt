# Do zero para programador front-end em 8 passos


1. HTML5 e CSS3

HTML
  - linguagem de marcação
  - estrutura página web
  - atribui sgnificado a cada elemento da página
  
CSS
  - linguagem de estilização
  - aplicada aos elementos HTML
  - atrativo estético

2. JavaScript
  - linguagem de programação
  - lógica de programação
  - funcionalidades suportadas pelo navegador

3. Bibliotecas
  - padronização
  - produtividade
  - soluciona problema específico
  - jQuery
    - biblioteca
    - facililta alterações do usuário

4. Mais JS
  - versão ES6
    - novos recursos
    - soluciona problemas corriqueiros

5. Frameworks
  - conjunto de bibliotecas
  - solução a problemas maiores / comlexos
  - React / Angular / Bootstrap

6. Pré-processador
  - SASS / LESS
  - compilador
  - traduz sintaxe rebustada
  - trabalha com variáveis / funções CSS
  - dá poder de linguagem de programação ao CSS
  - torna trabalho mais produtivo

7. Ferramenta de build
  - site pronto
  - passa por série de otimizações
  - ferramentas utilizam JS
  - Gulp - ferramenta de build

8. Noções d back-end
  - funcionamento básico
  - integração dofront com back
  - linguagem PHP (back / páginas dinãmicas)

Conclusão

  - o básico abre oportunidades de trabalho
  - dominar bibliotecas, frameworks, noções de back
    - ampliam / fortalecem oportunidades de trabalho
